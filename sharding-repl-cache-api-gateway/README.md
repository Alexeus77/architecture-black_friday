# pymongo-api

## Как запустить

Запускаем кластер mongodb, redis, инстансы приложения, apisix с consul

```shell
docker compose up -d
```

Инициализируем кластр Mongo, заполняем данными, настраиваем apisix
(Скрипт для Windows)

```shell
./scripts/init-mongo.cmd
```

## Как проверить

### Если вы запускаете проект на локальной машине

Откройте в браузере http://localhost:9080

Запросы пойдут на инстансы приложения по round robin

### Если вы запускаете проект на предоставленной виртуальной машине

Узнать белый ip виртуальной машины

```shell
curl --silent http://ifconfig.me
```

Откройте в браузере http://<ip виртуальной машины>:9080

## Доступные эндпоинты

Список доступных эндпоинтов, swagger http://<ip виртуальной машины>:9080/docs
